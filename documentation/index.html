Edit Text Top

Open index.html to change all static texts. For example, to change the text "Coming soon. Stay Tuned." edit text between <h1></h1> tags on line 36.

Change Colors Top

Replace the html color values in color.css to change colors. For example, to change the main color to a nice green color, replace #D45661 with #19DD89. 

a, a:hover, a:focus {
	color: #19DD89;
}

To change the color of the countdown circles please open catapult.js within the js folder and edit line 22-27. 

Hours: { color: '#D45661' },

Below are some colors with the corresponding values - that we think fits nicely with the template:

  • #19DD89
  • #0598F0
  • #FFB237

You can find more html color values at http://www.w3schools.com/html/html_colorvalues.asp.

Replace Logo Top

To change the logo replace logo.png which can be found in the images folder with your own logo.

If you have used a different name or extension for the logo, for example company.jpg then you have to change the source of the img tag in index.html on line 32.

<img src="images/company.jpg" alt="Your Company Name">

If you don't want a logo, just replace the img tag with text.

Change the launch date and time Top

Edit the data-date value in index.html on line 46 to change the launch date and time. 

<div class="catapult-countdown" data-date="2015-01-01 00:00:00"></div>

The format of the value is yyyy-mm-dd hh:mm:ss (year-month-day hours:minutes:seconds)

Replace Background Slider Images Top

The images used for the background can be found in the photos folder within the images folder. Replace it with your own images or add new images. Change their names in index.html on line 108-110 or add more if necessary. 

<ul id="catapultBG" class="catapultImages">
	<li><img src="images/blank.png" style="background-image: url(images/photos/01.jpg);"></li>
	<li><img src="images/blank.png" style="background-image: url(images/photos/02.jpg);"></li>
	<li><img src="images/blank.png" style="background-image: url(images/photos/03.jpg);"></li>
</ul>

Single Image Background Top

Add only one image if you don't want a slider.

<ul id="catapultBG" class="catapultImages">
	<li><img src="images/blank.png" style="background-image: url(images/photos/01.jpg);"></li>
</ul>

Modify Social Profiles Icons Top

To modify the social profile icons edit index.html on line 88-92. Edit de class of the i tag to change the icons and place your link url in the href attribute. 

<a href="LinkToYourProfile" title="Twitter"><i class="fa fa-twitter"></i></a>
